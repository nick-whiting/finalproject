    Steps for Installing Distros

    1. Download the ISO file for the desired Linux distribution.
    2. Create a bootable USB drive using tools like Rufus or Etcher.
    3. Boot from the USB drive and start the installation process.
    4. Follow the on-screen instructions to partition the hard drive.
    5. Complete the installation and reboot the system.
    6. Repeat the process for each distribution you want to install.
